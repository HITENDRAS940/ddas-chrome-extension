# Chrome Extension Icons

The DDAS extension requires icons in the following sizes:
- 16x16 pixels (toolbar icon)
- 48x48 pixels (extension management page)
- 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create these icons using any graphic design tool. Here's a simple approach:

### Option 1: Using Online Tools
1. Go to https://www.favicon-generator.org/
2. Upload a logo or create one
3. Download icons in required sizes
4. Rename them to icon16.png, icon48.png, icon128.png

### Option 2: Using Photoshop/GIMP
1. Create a new image with purple/blue gradient background
2. Add "DDAS" text or a file/search icon
3. Export in 16x16, 48x48, and 128x128 sizes

For now, the extension will work without icons, but Chrome will show a default placeholder.

## Recommended Design
- Background: Purple gradient (#667eea to #764ba2)
- Icon: White magnifying glass or file icon
- Text: "DDAS" in white, bold font
- Style: Modern, rounded corners, flat design
